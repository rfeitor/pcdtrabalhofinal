NAME
     projectInfo -- identify build tool for software development project

SYNOPSIS
     projectInfo key [key ...] [-inDirectory path]

DESCRIPTION
     This tool is used to identify the build tool, such as /usr/bin/xcodebuild
     or /usr/bin/gnumake, for a software development project.  The use of this
     tool is discouraged.

     Typical use of this tool is: projectInfo BUILDTOOL
