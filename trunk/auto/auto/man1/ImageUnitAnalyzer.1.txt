NAME
     ImageUnitAnalyzer -- Tool to verify Image Units to be used with Core
     Image.

SYNOPSIS
     ImageUnitAnalyzer [ImageUnit]

DESCRIPTION
     Use the ImageUnitAnalyzer to verify that your ImageUnit follows all the
     requirements set by Core Image.

     A list of items with descriptions:

     ImageUnit
              File path of the bundle of the Image Unit
